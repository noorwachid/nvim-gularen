# Gularen Toolkit

## Features
- `kode` text-object
- `checklist` toggler

### `kode` text-object
Can be accessed using `ik` and `ak`

### `checklist` toggler
Can be accessed using `:GularenTKDo` and `:GularenTKCancel`.
Or ranged version using `:'<,'>GularenTKDoRange` and `:'<,'>GularenTKCancelRange`.
